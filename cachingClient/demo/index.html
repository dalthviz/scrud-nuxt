index.js

Use the devtools of the browser to interact with the cachingClient

For example you can try in the console:

client = new cachingClient.default(); await client.get("https://jsonplaceholder.typicode.com/users/1/todos", true);

To check a dummy JSON of a list of todos.

Also, you can use:

await client.clearCache();

To clear/empty the cache.
